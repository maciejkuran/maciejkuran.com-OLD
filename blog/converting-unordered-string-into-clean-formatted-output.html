Converting an unordered, long string into a clean formatted output using string methods

15-05-2022 JavaScript Exercise

In this exercise, I am converting an unordered, very long string into a clean formatted output. I will retrieve all necessary data dynamically without hard-coding using available string methods.

I am working on this string.

            const flights =
  '_Delayed_Departure;mao25678109;bon1463757340;12:30+_Arrival;lon0945634522;mon93764345;20:45+_Delayed_Arrival;ams2136549680;bra100766409;18:15+_Departure;syd10564306;por564343677;20:30+_Arrival;lon10444376;war3333567;23:30';

Final output expected.

游댮  Delayed Departure from MAO to BON (12h30) 
            游릭  Arrival from LON to MON (20h45) 
            游댮  Delayed Arrival from AMS to BRA (18h15) 
            游릭  Departure from SYD to POR (20h30) 
            游릭  Arrival from LON to WAR (23h30)

Final code.

            const flightsInfo = string => {
                const stringToArray = string.split('+');
                console.log(stringToArray);
                
                for (let i = 0; i < stringToArray.length; i++) {
                  // console.log(stringToArray[i]);
                  const [status, from, to, time] = stringToArray[i].split(';');
                  //destructuring array and saving values to seperate variables
                  // console.log(status, from, to, time);
              
                  console.log(`${status.includes('Delayed') ? '游댮' : '游릭'} ${status.trim().replaceAll('_', ' ')} from ${from.slice(0, 3).toUpperCase()} to ${to.slice(0, 3).toUpperCase()} (${time.replaceAll(':', 'h')})`);
                }
              };
              
              flightsInfo(flights);
