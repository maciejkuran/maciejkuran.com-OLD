Remainder operator

There's a rule that says that every even number can be divided by 2. Let's check if the numbers I pass as arguments are even, using the remainder operator - syntax %. 

              
                  const checkIfEven = n => n % 2 === 0;
                  console.log(checkIfEven(6)); //OUTPUT: true
                  console.log(checkIfEven(9)); //OUTPUT: false
                  console.log(checkIfEven(22)); //OUTPUT: true
