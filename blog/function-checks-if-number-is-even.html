The function that checks if a certain number is even

18-06-2022 JavaScript Exercise

There's a rule that says that every even number can be divided by 2. Let's check if a certain number is even using the remainder operator. 

            const checkIfEven = n => n % 2 === 0;

console.log(checkIfEven(6)); //OUTPUT: true
console.log(checkIfEven(9)); //OUTPUT: false
console.log(checkIfEven(22)); //OUTPUT: true
