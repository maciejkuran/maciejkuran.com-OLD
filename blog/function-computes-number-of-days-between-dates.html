The function that computes the number of days between a certain dates

18-06-2022 JavaScript Exercise

In this exercise, I am writing the function that computes the number of days between certain dates. Let's assume, there are 3 events in January. The dates have already been announced and stored in an array, in the object 'january'. 

            const january = {
                events: [
                  '2023-01-01T19:30:00.000Z',
                  '2023-01-10T20:30:00.000Z',
                  '2023-01-30T21:30:00.000Z',
                ],
              };
            
              
            const calcDays = (date1, date2) =>
  Math.round(
    Math.abs(+new Date(date1) - +new Date(date2)) / (1000 * 60 * 60 * 24)
  );

          
//Calculating the days between the 1st and 2nd event
            console.log(calcDays(january.events.at(0), january.events.at(1))); //9 DAYS
            
            //Calculating the days between the 1st and 3rd event
            console.log(calcDays(january.events.at(0), january.events.at(2))); // 29 DAYS
