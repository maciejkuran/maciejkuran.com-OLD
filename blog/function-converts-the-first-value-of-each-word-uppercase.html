Converting the first value of each word to uppercase

Array and string methods

This function converts a string to lowercase and then converts the first value of each word to uppercase - in a string passed as an argument when the function is called.

This function is helpful if you e.g. need to convert a name and last name and have it printed in DOM. It's also nice for specific names like books titles or building names.

The function uses such methods as:

  • split() - it allows to divide a string into multiple part based on divider string. The elements are saved in an array.
  • toLowerCase() - it converts values to lowercase.
  • toUpperCase() - it converts values to uppercase.
  • slice() - it 'cuts' a string.
  • join() - the opposite of split. It converts back array elements to a string.
  • push() - the push method appends values to an array.
            
                const capitalizeName = name => {
                    const names = name.split(' ');
                    const namesUpper = [];
                    for (const n of names) {
                    const toLowerCase = n.toLowerCase();
                    namesUpper.push(toLowerCase[0].toUpperCase() + toLowerCase.slice(1));
                    }
                    
                    const convertedName = namesUpper.join(' ');
                    
                    return convertedName;
                    };
                
            
        
            
                console.log(capitalizeName('jOHn kowALSKI')); //OUTPUT: John Kowalski
console.log(capitalizeName('the museum of science and industry, chicago'));
//OUTPUT:The Museum Of Science And Industry, Chicago
