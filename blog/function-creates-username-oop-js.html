Constructor function and prototype

In this exercise, I am writing the function that generates a username for each user's object. This exercise highlights the Object Oriented Programming paradigm (OOP) in JavaScript. 

              
                  const User = function (name, lastName) {
                      this.name = name;
                      this.lastName = lastName;
                    };
        
                    //Creating 2 objects - 'User' instances
                    const user1 = new User('Maciej', 'Kuran-Janowski');
                    const user2 = new User('John', 'Kowalski');
        
              
          
              
                  //Creating a prototype that is linked to 'User'
        User.prototype.username = function () {
          const username =
            this.name.toLowerCase().slice(0, 3) +
            this.lastName.toLowerCase().slice(0, 3);
          console.log(username);
        };
        
        user1.username(); //mackur
        user2.username(); // johkow

ES6 Classes

Instead of creating a classic constructor function, we may do it in a modern way using ES6 Classes. Below, I am rewriting the code. 

              
                  const User = class {
                      constructor(name, lastName) {
                        this.name = name;
                        this.lastName = lastName;
                      }
        
                      username() {
                        const username =
                          this.name.toLowerCase().slice(0, 3) +
                          this.lastName.toLowerCase().slice(0, 3);
                        console.log(username);
                      }
                    };
        
        
              
          
              
                  const user1 = new User('Maciej', 'Kuran-Janowski');
                  const user2 = new User('John', 'Kowalski');
        
                  user1.username(); //mackur
                  user2.username(); //johkow
