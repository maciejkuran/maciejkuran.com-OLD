The function that masks credit card digits, showing only the last 4

15-05-2022 JavaScript Exercise

In this exercise, I am working with string methods. I wrote the function that masks credit card digits, leaving the last 4.

I used the slice() and padStart() string methods.

slice() – cuts a string based on argument.

padStart() – the 1st argument defines how many characters does a string must have and 2nd defines the value that a string will be completed with. 

            //sample primitive string created, but the Function works with any argument passed

const creditCard = '1234567891011123';
console.log(creditCard.length); //16 digits

const hidingDigits = digits => {
  digits.trim();
  const slicedDigits = digits.slice(-4); //last 4 digits
  const hidingDigits = slicedDigits.padStart(digits.length, '*'); //1st argument - string characters quantity, 2nd - value that the string will be completed with

  return hidingDigits;
};

console.log(hidingDigits(creditCard)); //OUTPUT: ************1123
console.log(hidingDigits('9999999999999999')); //OUTPUT: ************9999
console.log(hidingDigits('12346666')); //OUTPUT: ****6666
