reduce method

In this exercise, I am writing an arrow function that returns the highest number from any array passed as an argument when we call the function.

There are plenty of concepts in JavaScript that can be used to solve this problem: e.g. for loop or for of loop. However, this time I want to use the reduce() method.

The reduce method reduces all array values to one single value, based on set condition.

I am creating a sample array. Let's find out what is the highest number.

            
                const array = [1, 5, 2, 7, 99, 10, 50, 555];
            
        
            
                const findHighestNumber = arr => {
                    const highestNumber = arr.reduce((previousNumber, number) => {
                    if (previousNumber > number) {
                    return previousNumber;
                    } else {
                    return number;
                    }
                    }, arr.at(0));
                    
                    return highestNumber;
                    };
                    
                    console.log(findHighestNumber(array)); //OUTPUT: 555

Another solution - sort and at methods

I find the sort() method helpful solving this problem. It makes this code look clean and easy to read. If you don't know how the sort() method works, you may dive deeper into the MDN documentation. 

            
                const array = [1, 5, 2, 7, 99, 10, 50, 555];
                const findHighestNumber = arr => {
                return arr.sort((a, b) => b - a).at(0);
                };
                
                console.log(findHighestNumber(array)); //OUTPUT: 555

Returning the smallest number using sort and at methods

Even though this isn't part of the exercise, to curiosity let's check how to return the smallest number from an array using the sort() and at() methods. 

            
                const array = [1, 5, 2, 7, 99, 10, 50, 555];
                const findHighestNumber = arr => {
                return arr.sort((a, b) => b - a).at(arr.length - 1);
                };
                
                console.log(findHighestNumber(array)); //OUTPUT: 1
