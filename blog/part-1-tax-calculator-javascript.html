Function returns another function

In this exercise, I am writing the function that returns another function. This concept is commonly used in the functional programming paradigm. However you don't really have to be interested in functional programming in order to use this concept. It can be helpful in many situations.

I am writing the function that will calculate any tax value.

The function taxCalc that takes a parameter rate is a higher-order function. Just a reminder, a higher-order function is the function that takes other functions as arguments, returns another function, or both.

Why do I think that this example is that important?

Actually, by returning a function from another function we gain some benefits. We can use the same function on and on to calculate any tax value. Let's dive deeper into this example.

I am writing the function that returns another function and does calculation.

            
                const taxCalc = rate => {
                    return function (value) {
                    const tax = (value * rate) / 100;
                    console.log(`Value: ${value}zl\nTax rate: ${rate}%\nYour tax: ${tax}zl`);
                    };
                    };

In Poland, there's a VAT tax. It is an additional value that is added to any goods that you buy. There are different rates but the main one is 23 %.

The currency in Poland is 'Zloty' – zl.

            
                const VATPoland = taxCalc(23);

I created the constant variable VATPoland and assigned a higher-order function, passing the number 23 as an argument – tax rate.

Now we are moving to the returned function that will be invoked. The number 100 is the value and, based on our example – the money. 

            
                VATPoland(100);
                //OUTPUT:
                // Value: 100zl
                // Tax rate: 23%
                // Your tax: 23zl

The function taxCalc() we can save to any variable anytime. I also want to calculate the VAT tax rate in Germany. The standard VAT rate in Germany is 19 %. 

            
                const VATGermany = taxCalc(19);
                VATGermany(100);
                //OUTPUT:
                // Value: 100zl
                // Tax rate: 19%
                // Your tax: 19zl

There's some other functional programming concept to write the tax calculator – using the partial application concept and the bind() method.
