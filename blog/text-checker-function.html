Text checker function – checking if the text includes forbidden words / characters - some() method

07-05-2022 JavaScript Exercise

This function checks if a certain text, e.g. submitted by the user includes forbidden words or characters.

If yes, a specific notification will be returned to a user.

const forbiddenWords = ['email', '@', 'tel'];

let notification = '';

const textChecker = (text, arr) => {
  const input = text.toLowerCase();
  if (arr.some(word => input.includes(word))) {
    notification = `Forbidden words: ${arr
      .join(' ')
      .replaceAll(' ', ', ')}`;
  } else {
    notification = 'Successfully submitted';
  }
  return notification;
};

              
const string1 = 'Hi there, How are you ?';
            console.log(textChecker(string1, forbiddenWords));
