The function that returns the sum of unlimited numbers passed as arguments

05-05-2022 JavaScript Exercise

I am writing the function that returns the sum of unlimited numbers passed as arguments when function is called, using the 'rest' parameter.

The 'Rest' can be used as a parameter if we do not know how many arguments will be passed when a function is called.
The documentation about 'rest' parameter is available here. 

            let sumResult = 0;

const calc = function (...numbers) {
  for (let i = 0; i < numbers.length; i++) {
    sumResult += numbers[i];
  }
  return sumResult;
};

console.log(calc(2, 2, 5, 1)); //OUTPUT: 10
