The function that checks the login email address for capital letters or spaces and removes them

05-05-2022 JavaScript Exercise

This function can be useful if it comes to work in the login area where you need to check user's email address and compare to the one that is already registered. A user can type email address with uppercase and also make unnecessary spacing. This function solves that problem. 

            //Function that compares / checks email

            const compareEmail = email => {
              const correctEmail = email.toLowerCase().trim(); //letters to lower case & removing spacing with 'trim' method
              return correctEmail;
            };
            
            console.log(compareEmail('    aDDreSS@.EmAil.COM')); //OUTPUT: address@email.com
