The function that returns corrected first name/last name in case the user unnecessary will capitalize some letters

05-05-2022 JavaScript Exercise

This function converts any name passed as an argument. The first letter of a string will be returned as the capital and all the other letters will be converted to lowercase. 

            const nameCorrect = function (name) {
                const toLowerCase = name.toLowerCase();
                const correctName = toLowerCase[0].toUpperCase() + toLowerCase.slice(1);
                return correctName;
              };
              
              console.log(nameCorrect('mACieJ')); //OUTPUT: Maciej
