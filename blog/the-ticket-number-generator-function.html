The ticket number generator function

15-05-2022 JavaScript Exercise

In this exercise, I am writing the function that will choose a random letter from the array, generate a random number and return a random letter + random number as a string. 

            const ticketNumberGenerator = function () {
                const letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h'];
                const randomLetter = Math.floor(Math.random() * letters.length);
                const numbers = Math.floor(Math.random() * 100000);
              
                return letters[randomLetter].toUpperCase() + numbers;
              };
              
              console.log(ticketNumberGenerator()); //OUTPUT H64912
